# Nooto
encrypted note-taking software  
User password hashed using scrypt, notes title and content encrypted using AES+CTR+HMAC.

## Requirements
* Python 2.6
* PyQt5
* pyscript (https://github.com/ricmoo/pyscrypt)
* tinymce 4 community

## running
`python main.py`
